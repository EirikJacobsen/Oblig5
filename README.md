# UtleieApp

Dette er en Flask-applikasjon for et utleiesystem basert på databasen fra Oblig 3.

## Teknologier
- Flask
- MySQL
- Flask-Login
- Jinja2
- CSS

## Hvordan kjøre appen
1. Opprett database i MySQL med oblig3.sql
2. Kjør login_tabell.sql
3. Opprett virtuelt miljø
4. Installer pakker:
   pip install -r requirements.txt
5. Start appen:
   python app.py

## Funksjoner
- Login/logout
- Dashboard
- Kundeliste
- Registrere kunde
- Liste over utstyr
- Registrere utleie
- Registrere innlevering
- Statistikkside
